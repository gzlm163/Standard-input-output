# Лабораторная работа: Стандартный ввод-вывод

## Автор
Гинжул Мария

## Цель
Разработка приложения для работы с текстовыми файлами, освоение механизмов сериализации, применение паттерна «Memento» для отмены изменений и реализация поиска файлов по ключевым словам.

## Описание
Проект представляет собой комплексное решение для управления текстовыми файлами. В его состав входят:
- Класс для представления текстового файла с поддержкой сериализации.
- Класс для поиска файлов в директории по ключевым словам.
- Консольный текстовый редактор с возможностью отката изменений.
- Механизм индексации для быстрого поиска файлов по содержимому.
